# 2. Paleta de Cores e Tipografia

## Objetivo

Definir e justificar as escolhas de cores e fontes que compõem a identidade visual do Jornada Verde. As decisões de design são baseadas em:
- **Psicologia das cores** e significado para o público-alvo (adolescentes)
- **Tipografia diferenciada** que comunica modernidade, não formalidade
- **Legibilidade** e coerência visual
- **Acessibilidade** em interface mobile-first

### 5. Estrutura de Cores: Claro → Médio → Escuro


**Por que essa abordagem?**
- Permite diferenciar estados (hover, active, disabled)
- Cria profundidade sem usar cores completamente diferentes
- Mantém identidade visual coerente
- Facilita legibilidade em diferentes contextos

**Tabela de Hierarquia:**

| Elemento | Cor | Uso |
|----------|-----|-----|
| **Cards principais** | Verde Claro #338637 | Destaque suave |
| **Botões primários** | Verde Médio #196124 | Ação principal |
| **Hover/Focus** | Verde Escuro #064E11 | Feedback visual |
| **Elementos ativos** | Verde Escuro #064E11 | Estado selecionado |
| **Texto secundário** | Cinza #A4A4A4 | Hierarquia |
| **Background** | Branco #FFFFFF | Leveza |

# Tipografia

### 1. Fonte Primária (Títulos): **Montserrat**

<img width="586" height="468" alt="image" src="https://github.com/user-attachments/assets/e0544236-545c-4064-8479-80cf646020a0" />

**Especificações:**
- **Família:** Montserrat
- **Peso:** Regular (padrão, sem peso específico)
- **Tamanhos recomendados:**
  - H1 (Títulos de página): **28-32px**
  - H2 (Títulos de seção): **24-26px**
  - H3 (Subtítulos): **18-20px**
  - Botões: **14-16px**

**Link:** [Google Fonts - Montserrat](https://fonts.google.com/specimen/Montserrat)

#### Justificativa

**Modernidade, não Formalidade**
- Montserrat é uma fonte **geométrica** e **contemporânea**
- Transmite "tech", "moderno", "agora" — exatamente o que adolescentes veem em apps que confiam
- Evita a rigidez de fontes muito formais (que adolescentes associam com escola/chato)

**Legibilidade em Mobile**
- Caracteres abertos e bem diferenciados
- Excelente em telas pequenas
- Peso regular é confortável para leitura, não muito fino nem muito pesado

**Identidade Visual Diferenciada**
- Montserrat é reconhecível e moderna
- Comunica que o Jornada Verde é um app DIFERENTE, não é "coisa de professor chato"
- Geométrica = organização, sustentabilidade, ordem

**Público-Alvo (Adolescentes):**
- Reconhecem em apps populares e confiáveis
- Não parece "corporativo" nem "infantil"
- Equilibra seriedade com leveza

### 2. Fonte Secundária (Corpo): **Roboto Medium**

<img width="548" height="289" alt="image" src="https://github.com/user-attachments/assets/2e0e62e7-b279-4821-8947-36d35b3c3a31" />

**Especificações:**
- **Família:** Roboto
- **Peso:** Medium (500)
- **Tamanhos recomendados:**
  - Corpo de texto: **14-16px**
  - Labels: **12-14px**
  - Help text / Captions: **11-13px**

**Link:** [Google Fonts - Roboto](https://fonts.google.com/specimen/Roboto)

#### Justificativa

**Legibilidade em Texto Longo**
- Roboto Medium é mais "amigável" que uma fonte muito fina
- Peso medium (não light, não bold) é perfeito para leitura contínua
- Caracteres bem espaçados naturalmente

**Harmonia com Montserrat**
- Ambas são sans-serif modernas (coerência visual)
- Roboto tem proporções clássicas enquanto Montserrat é geométrica = contraste interessante
- Funcionam bem juntas sem parecer "bagunçado"

**Mobile-First**
- Roboto foi desenhada pelo Google para Android (responsiva por natureza)
- Medium weight garante legibilidade mesmo em telas pequenas sem parecer pesado
- Bom espaçamento de caracteres

**Acessibilidade**
- Roboto Medium é legível para dislexia
- Contraste alto com branco é excelente
- Peso medium é melhor que light para leitura prolongada

## Coerência com o Público-Alvo

### Para Adolescentes

✅ **Tipografia moderna, não formal**
- Montserrat + Roboto = contemporâneo, não corporativo
- Comunica "isso é feito para gente como eu"

✅ **Cores vibrantes mas confiáveis**
- Verde #196124 é vivo, não pastoso
- Múltiplos tons criam dinamismo sem caos

✅ **Mobile-first design**
- Tamanhos recomendados (14-16px corpo) são confortáveis em celular
- Fontes desenhadas para tela

---

### Para Professores

✅ **Profissionalismo**
- Verde comunica responsabilidade (ambiental e educacional)
- Tipografia clara é essencial para explicar conceitos

✅ **Legibilidade**
- Roboto Medium em corpo, títulos em Montserrat = hierarquia clara
- Funciona bem em projetor (grande contraste)

---

### Para Acessibilidade

✅ **Contraste garantido**
- Verde #196124 + Branco = 7.2:1 ✅
- Verde #064E11 + Branco = 9.1:1 ✅ (excelente!)
- Preto + Branco = 21:1 ✅ (perfeito)

✅ **Fontes inclusivas**
- Montserrat e Roboto têm desenho acessível
- Caracteres bem diferenciados (evita confusão)

✅ **Sem dependência de cor**
- Verde não é a ÚNICA forma de comunicar algo
- Sempre acompanhado de ícones, texto, símbolos
